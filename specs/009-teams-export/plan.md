# Implementation Plan: Teams Export — Backend Endpoint

**Branch**: `009-teams-export` | **Date**: 2026-05-06 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `specs/009-teams-export/spec.md`

## Summary

Create a `GET /export/teams?eventId={uuid}&format=xlsx|csv` REST endpoint that
returns a deduplicated, formatted export of all teams enrolled in a competition.
The controller lives directly in the API app alongside `CpController` — no new
Nx lib created. The existing enrollment endpoint is out of scope and untouched.

The endpoint applies Dutch day-name translation and `HH:mm` time formatting —
improvements over the old Angular client which emitted raw ENUM strings.

## Phase 0: Research Findings

See [research.md](research.md) for full decision log. Key decisions:

1. **Auth pattern**: Two-step inline check — `UnauthorizedException` (401) for no auth,
   `ForbiddenException` (403) for no permission. Established in spec 008.

2. **Format switching**: `format` query param (`xlsx` | `csv`, default `xlsx`). Controller
   validates, calls service, then calls `toXlsx` or `toCSV` from `@badman/backend-utils`.
   Services never touch serialization.

3. **Module placement**: `ExportController` lives directly in the API app
   (`apps/api/src/app/controllers/`), registered in `AppModule.controllers` like
   `CpController`. No new Nx lib. Enrollment lib not touched.

4. **Deduplication**: `Set<string>` of team IDs during traversal — O(1), no extra queries.

5. **Day name translation**: ENUM → Dutch (`monday` → `Maandag`, etc.) inline map in
   `TeamsService`. Fixes the inconsistency in the old Angular client.

6. **Time formatting**: `"HH:mm:ss"` → `"HH:mm"` via string slice. Missing → `""`.

## Phase 1: Design & Contracts

### ExportController

Single controller at path `export`. Each route handler follows this guard order:

```
1. if (!user?.id)                     → UnauthorizedException (401)
2. if (!hasPermission)                → ForbiddenException   (403)
3. if (format not in ['xlsx','csv'])  → BadRequestException  (400)
4. if (!IsUUID(eventId))              → BadRequestException  (400)
5. EventCompetition.findByPk → null  → NotFoundException    (404)
6. call service.getTeams(eventId)
7. toXlsx or toCSV based on format
8. set Content-Type + Content-Disposition, send
```

### TeamsService

```typescript
getTeams(eventId: string): Promise<{
  headers: readonly string[];
  rows: (string | number | undefined | null)[][];
  eventName: string;
}>
```

- Traverses `EventCompetition → SubEventCompetition → DrawCompetition → EventEntry → Team → Club`
- Deduplicates by `team.id` using a `Set<string>`
- Translates `preferredDay` to Dutch; formats `preferredTime` as `HH:mm`
- Returns raw `{ headers, rows, eventName }` — no XLSX/CSV logic
